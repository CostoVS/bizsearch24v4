## A52 Environment Git & Pull Build Recovery Info

This section tracks build constraints for pulling and building the repository on local environments (such as terms referenced in A52 mobile/local syncs or git updates). If local git pulls encounter build issues or package discrepancies, run the following recovery routine:

1. **Clean build artifacts & cache:**
   ```bash
   rm -rf .next node_modules package-lock.json
   ```
2. **Re-install clean client dependencies:**
   ```bash
   npm install
   ```
3. **Execute Production Compile:**
   ```bash
   npm run build
   ```
